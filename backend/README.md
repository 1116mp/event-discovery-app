# Backend - Event Discovery API

Express.js REST API for the Event Discovery application.

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

Server runs on `http://localhost:3001`

## API Endpoints

### POST /api/events
Create a new event.

### GET /api/events
List all events with optional location filter and distance calculation.

### GET /api/events/:id
Get event details by ID.
