# Frontend - Event Discovery App

React frontend for the Event Discovery application, built with Vite.

## Installation

```bash
npm install
```

## Running

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Building for Production

```bash
npm run build
```

The production build will be in the `dist/` directory.

## Preview Production Build

```bash
npm run preview
```
